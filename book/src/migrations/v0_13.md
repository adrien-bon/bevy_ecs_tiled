# Migration to `bevy_ecs_tiled` v0.13

This guide outlines the steps required to migrate your project to version `0.13` of `bevy_ecs_tiled`.

## Main Changes

- **Bevy v0.19 Support:**
  `bevy_ecs_tiled` now works seamlessly with Bevy v0.19, allowing you to take advantage of the latest engine features and performance improvements. See the [Bevy 0.19 release notes](https://bevy.org/news/bevy-0-19/) for details.

## Migration Steps

1. **Update Dependencies in `Cargo.toml`:**

    ```toml
    bevy = "0.19"
    bevy_ecs_tiled = "0.13"
    ```

    > **Note:** You may also need to update other Bevy-related crates to ensure compatibility.

2. **Review the [Bevy Migration Guide](https://bevy.org/learn/migration-guides/0-18-to-0-19/):**
   Follow Bevy's official migration steps to update your project for Bevy v0.18.
   This will help you resolve breaking changes and take advantage of new features.
