# Migration to `bevy_ecs_tiled` v0.12

This guide outlines the steps required to migrate your project to version `0.12` of `bevy_ecs_tiled`.

## Main Changes

- **Physics colliders rework**

## Migration Steps

1. **Update Dependencies in `Cargo.toml`:**

    ```toml
    bevy = "0.18"
    bevy_ecs_tiled = "0.12"
    ```

    > **Note:** If you were using Avian, this dependency was also updated to v0.6
